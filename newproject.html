AWS Services Used

Amazon S3

Hosts the static resume website.

CloudFront

Provides CDN and HTTPS security.

Lambda

Handles visitor counter logic.

DynamoDB

Stores visitor count securely.

API Gateway

Connects frontend with backend.

GitHub Actions

Automates CI/CD pipeline.

Architecture

Our Cloud Resume Challenge uses a serverless AWS architecture where a static website is hosted on Amazon S3 and delivered using CloudFront. A Lambda function integrated with API Gateway updates visitor count stored in DynamoDB.

Core Features

Static Website Hosting

Resume website is hosted on Amazon S3 with high availability and scalability.

Serverless Backend

Backend logic is implemented using AWS Lambda without managing servers.

Visitor Counter

DynamoDB stores visitor count which is updated through API Gateway and Lambda.

Global Content Delivery

CloudFront ensures fast and secure delivery of the website worldwide.

CI/CD Automation

GitHub Actions automatically deploys frontend and backend code.

Secure & Scalable

IAM roles, HTTPS, and serverless services ensure security and scalability.
